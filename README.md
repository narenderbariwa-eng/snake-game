# snake-game
 # 🐍 Snake Game (React)

## 📌 Project Description

Snake Game is a simple and interactive web game developed using **React, HTML, CSS, and JavaScript**. In this game, the player controls a snake that moves around the screen to eat food and grow longer. The objective is to score as many points as possible without colliding with the wall or the snake’s own body.

## 🚀 Features

* Interactive Snake Game
* Real-time score tracking
* Keyboard controls for movement
* Simple and responsive design
* Game over detection

## 🛠️ Technologies Used

* React.js
* HTML5
* CSS3
* JavaScript

## 🎮 How to Play

1. Start the game.
2. Use the **Arrow Keys (↑ ↓ ← →)** to control the snake.
3. Eat the food to increase the score.
4. Avoid hitting the walls or the snake’s body.
5. Try to achieve the highest score.

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/snake-game.git
```

2. Go to the project folder

```
cd snake-game
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm start
```

The project will run on **http://localhost:3000**
